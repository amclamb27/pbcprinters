Printer Setup

Adds all PBC office printers to your laptop in about 30 seconds. Pick your operating system below.

macOS — Install Printers

  1. Open Terminal (Spotlight: ⌘+Space, type Terminal, press Enter).
  2. Click Copy on the command below.
  3. Paste into Terminal (⌘+V) and press Enter.
  4. Type your Mac password when prompted, then press Enter. (The password won't appear as you type — that's normal.)
  5. Wait about 10 seconds. You'll see "All Providence Baptist Church printers are now available."
curl -fsSL https://raw.githubusercontent.com/amclamb27/pbcprinters/main/mac/install-pbc-printers.sh | sudo bash
That's it. Open any document → File → Print and pick a Toshiba printer from the list.

Windows — Install Printers

  1. Press the ⊞ Win key, type PowerShell.
  2. Right-click "Windows PowerShell" → Run as administrator.
  3. Click Yes on the User Account Control prompt.
  4. Click Copy on the command below, paste it into PowerShell (right-click in the window to paste), and press Enter.
  5. Wait about 30 seconds. You'll see green "ADDED" lines for each printer.
irm https://raw.githubusercontent.com/amclamb27/pbcprinters/main/windows/Install-PBCPrinters.ps1 | iex
Must be run as Administrator. If you see a red "Access denied" or "must be run as Administrator" message, you missed step 2 — close PowerShell and start over with "Run as administrator".
